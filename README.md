# N&H - Notre Amour ❤️

Application web pour couple à distance — messagerie, galerie, carte, dessin, jeux.

## Quick start

```bash
npm install
npm run dev
```

## Déploiement

Push sur `master` → déploiement auto sur Netlify.

```
git push origin master
```
