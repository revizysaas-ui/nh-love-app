# N&H - Notre Amour · Récap Projet

## Objectif
App web pour couple à distance (N & H) : messagerie, galerie photo, carte, dessin partagé, jeux.
Backend Supabase, déploiement Netlify.

---

## Base de données (Supabase)

### Tables
| Table | Utilité | Politique RLS |
|-------|---------|---------------|
| `rooms` | Salons (code, noms, dates, villes, coords) | PUBLIC (toutes ops) |
| `messages` | Messages texte + image_url | PUBLIC |
| `photos` | Galerie photos (storage_path) | PUBLIC |
| `drawings` | Dessins sauvegardés (data_url) | PUBLIC |
| `questions` | Questions Vérité/Action (80+ questions) | PUBLIC |
| `notifications` | Notifications d'activité | PUBLIC |

### Colonnes spéciales de `messages`
- `text TEXT` — le message
- `image_url TEXT` — URL de l'image envoyée (optionnelle)

### Storage
- Bucket `photos` — stocke les images (accès public)

### Migration à exécuter si nouvelle instance
```sql
ALTER TABLE messages ADD COLUMN IF NOT EXISTS image_url TEXT;

CREATE TABLE IF NOT EXISTS notifications (
  id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  room_id BIGINT REFERENCES rooms(id) ON DELETE CASCADE,
  type TEXT NOT NULL,
  message TEXT NOT NULL,
  author TEXT DEFAULT '',
  read BOOLEAN DEFAULT false,
  created_at TIMESTAMPTZ DEFAULT now()
);
ALTER TABLE notifications ENABLE ROW LEVEL SECURITY;
CREATE POLICY "Public notifications" ON notifications FOR ALL USING (true) WITH CHECK (true);
```

## Logique métier importante

### Salon (Room)
- Pas d'auth : RLS public sur toutes les tables
- Code 6 lettres majuscules, stocké dans `localStorage('nh_room')`
- 2 personnes partagent le même `room_id` → toutes les données filtrées par `room_id`

### Nom d'utilisateur
- Saisi UNE FOIS sur la page Join
- Stocké dans `localStorage('nh_username')`
- Utilisé partout (Messages, Dessin, Galerie) → plus besoin de le retaper

### Messages
- Realtime via Supabase channel `messages-{roomId}`
- L'utilisateur voit ses messages à droite (bulles `own`) et ceux de l'autre à gauche
- Les photos envoyées dans Messages sont aussi sauvegardées dans la table `photos` (galerie)

### Notifications
- Table `notifications` avec realtime
- Quand un message/photo/dessin est créé → `notify(roomId, type, message, author)`
- Toast affiché dans l'app (4s)
- Si la page est en arrière-plan → notification système (Desktop)

### Dark Mode
- Toggle dans la barre du bas (bouton ☀️/🌙)
- Classe `.dark` sur `<body>`, persisté dans `localStorage('nh_dark')`

### Palette
- Primaire : `#8a79ab` (violet)
- Accent : `#e6a5b8` (rose)
- Fond clair : `#f8f7fa`
- Fond sombre : `#1a1823`

### Responsive
- 3 breakpoints : défaut (mobile-first), ≤768px, ≤480px
- Sidebar désactivée, uniquement bottom nav
- Adapté téléphone + tablette + desktop

### PWA
- `manifest.json` → propose "Ajouter à l'écran d'accueil"
- Service worker (`sw.js`) → cache les assets pour mode offline partiel
- Icônes SVG en 192×192 et 512×512

### Déploiement
- `netlify.toml` : build `npm run build`, publish `dist`
- SPA redirect : `/* → /index.html`
